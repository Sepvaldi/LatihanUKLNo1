# LatihanUKLNo1
Fungsinya adalah untuk menghitung total biaya pengiriman berdasarkan berat paket jarak tempuh serta volume barang yang diketahui

Cara kerjanya yaitu user di minta untuk memasukkan angka oleh system, setelah user memasukan output yang diminta seperti berapa jarak tempuh pengiriman, berat barang, dsb. Maka system akan menerima output dari user lalu di proses oleh system seperti jarak tempuh pengiriman barang yang melebihi 10km maka akan di kenakanakan biaya tambahan sebesar Rp.6000 jika kurang atau sama dengan 10km maka dikenakan biaya 4250 dan volume barang juga akan dikenakan biaya tergantung bertanya, setelah itu system akan menghitung dan menampilkan total biaya yang akan di bayar oleh user.
